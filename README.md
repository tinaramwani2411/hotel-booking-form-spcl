🏨 Godwin Hotel – Booking Form

A responsive and interactive hotel booking form developed as part of internship training.
This project demonstrates form validation, responsive UI design, and dynamic user interaction using HTML, CSS, and JavaScript.

📌 Project Overview

The Godwin Hotel Booking Form allows users to:

Select check-in and check-out dates

Choose room type (Single, Double, Suite)

Enter number of guests

Add special requests

Submit booking with proper validation

The project focuses on frontend development, form handling, and UI/UX improvements.

✨ Features

✅ Responsive design (Mobile, Tablet, Desktop)

✅ Background image with dark overlay

✅ Glassmorphism luxury card design

✅ Premium gold theme styling

✅ Date validation (Check-out must be after Check-in)

✅ Restriction of past dates

✅ Dynamic error & success messages

✅ Tooltip guidance for certain fields

✅ Smooth fade-in and floating animations

🛠 Technologies Used

HTML5 – Structure of the form

CSS3 – Styling, animations, responsiveness

JavaScript (Vanilla JS) – Form validation & dynamic behavior

Git & GitHub – Version control

Netlify / GitHub Pages – Deployment

🎯 Validation Logic

Past dates are disabled.

Check-out date cannot be earlier than or equal to check-in date.

Required fields must be filled before submission.

Dynamic feedback messages guide the user.
